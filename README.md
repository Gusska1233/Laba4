# Задание 1

## Задача 1

### Текст задачи

Дерево содержит строки. Заменить в каждой строке каждый символ на следующий по порядку

### Алгоритм решения

Программа начинает работу с вывода заголовка "Бинарное дерево", после чего запрашивает у пользователя количество элементов в дереве, и для этого используется рекурсивная функция checkDigit, которая обеспечивает корректный ввод только положительных целых чисел, проверяя введенное значение с помощью TryParse и в случае ошибки или отрицательного числа повторяет запрос. Затем аналогичным образом запрашивается минимальная длина слова и максимальная длина слова. После получения всех необходимых числовых параметров вызывается функция createStringWord, которая принимает количество элементов, минимальную и максимальную длину, и с помощью генератора случайных чисел создает список строк указанной длины, где каждая строка состоит из случайных строчных букв латинского алфавита от a до z, причем для генерации символов используются ASCII коды от 97 до 122. Сгенерированный список строк выводится на экран. Далее с помощью функции insert и свертки List.fold создается бинарное дерево поиска, где каждая строка вставляется в соответствии с правилами: если строка меньше значения в узле, она идет в левое поддерево, если больше - в правое, а равные значения не добавляются. Полученное дерево выводится на экран с помощью функции printTree, которая выполняет рекурсивный обход дерева и печатает его элементы с отступами, отображая правые поддеревья выше, а левые ниже для визуального представления структуры. Затем вызывается функция shiftWord, которая применяется к исходному списку строк и с помощью List.map и String.map проходит по каждому символу каждой строки, заменяя его на следующий по порядку, при этом для символа 'z' выполняется особое условие - он заменяется на 'a', а для всех остальных символов просто увеличивается их ASCII код на единицу. Полученный список преобразованных строк выводится на экран, после чего из этого списка снова создается бинарное дерево поиска с помощью той же функции insert и List.fold, и это новое дерево также выводится на экран.

## Задача 2

### Текст задачи

Сформировать список из узлов с двумя листьями (узел является листом, если у него нет ни левого, ни правого поддерева).

### Алгоритм решения

Программа начинает работу с вывода заголовка "Бинарное дерево", после чего запрашивает у пользователя количество элементов в дереве, и для этого используется рекурсивная функция checkDigit, которая обеспечивает корректный ввод только положительных целых чисел, проверяя введенное значение с помощью TryParse и в случае ошибки или отрицательного числа повторяет запрос. После получения количества элементов вызывается функция createDigit, которая принимает это число и с помощью генератора случайных чисел создает список целых чисел в диапазоне от 1 до 15. Сгенерированный список чисел выводится на экран как "Сгенерированные числа:". Далее с помощью функции insert и свертки List.fold создается бинарное дерево поиска, где каждое число вставляется в соответствии с правилами: если число меньше значения в узле, оно идет в левое поддерево, если больше - в правое, а равные значения не добавляются, причем функция insert реализована таким образом, что при равенстве значений элемент просто не вставляется, сохраняя существующее дерево. Полученное дерево выводится на экран с помощью функции printTree, которая выполняет рекурсивный обход дерева и печатает его элементы с отступами, отображая правые поддеревья выше, а левые ниже для визуального представления структуры, что позволяет наглядно увидеть иерархию чисел в дереве. Затем определяется функция isLeaf, которая проверяет, является ли узел листом, возвращая true только для узлов, у которых есть значение, но нет ни левого, ни правого потомка, при этом пустое дерево Nil не считается листом. После этого вызывается основная функция nodesWithTwoListya, которая рекурсивно обходит все дерево и собирает в список те узлы, у которых оба потомка являются листьями, причем для этого используется вспомогательная функция collect с аккумулятором, которая сначала проверяет текущий узел на условие isLeaf left && isLeaf right, и если условие выполняется, добавляет значение узла в аккумулятор, затем рекурсивно обходит левое поддерево, передавая ему уже обновленный аккумулятор, а потом обходит правое поддерево, при этом в конце выполняется реверс списка, чтобы восстановить порядок обхода. Полученный список узлов выводится на экран как "Список узлов с 2 листами".
